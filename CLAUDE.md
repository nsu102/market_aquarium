기획안: Market Aquarium
AI 투자자들이 사는 가상 시장에 이벤트를 던지고, 공포와 탐욕이 어떻게 퍼져 가격·게시물·매매를 바꾸는지 관찰하고 참여하는 투자 심리 시뮬레이션.

핵심 구조
가상 시장 안에는 여러 AI 에이전트가 존재한다.
각 에이전트는 서로 다른 성격을 가진다.
패닉셀 개미
FOMO 단타러
가치투자자
음모론 인플루언서
퀀트 트레이더
매크로 고래
뉴스 요약 봇
반대로 투자하는 역발상 투자자
등등
이들은 이벤트가 발생하면 각자 판단해서 매매하고, 게시물을 올리고, 서로의 글에 영향을 받는다.

이벤트 방식
이벤트는 두 가지 방식으로 발생한다.
1. 사용자 생성 이벤트
사용자가 직접 입력한다.
예:
“트럼프가 중국 반도체 관세를 예고했다.”
“중동에서 전쟁 가능성이 커졌다.”
“비트코인 ETF 승인 루머가 돌았다.”
“대형 거래소 해킹 소식이 퍼졌다.”
2. 자동 생성 이벤트
시스템이 시장 상황에 맞게 자동으로 이벤트를 만든다.
예:
“유명 투자자가 특정 종목을 언급했다.”
“갑작스러운 금리 인하 기대감이 확산됐다.”
“실적 발표를 앞두고 루머가 증가했다.”
“커뮤니티에서 특정 코인이 밈화되기 시작했다.”

사용자의 역할
사용자는 세 가지 방식으로 참여한다.
1. 관찰자
시장 안에서 무슨 일이 벌어지는지 본다.
누가 먼저 반응했는지
누가 공포를 퍼뜨렸는지
어떤 에이전트가 대중을 따라갔는지
누가 반대로 움직였는지
가격이 감정에 의해 얼마나 왜곡됐는지

2. 투자자
사용자도 직접 매수·매도할 수 있다.
단순히 차트를 보는 게 아니라, 에이전트들의 게시물과 행동을 보고 판단한다.
예:
“개미들은 패닉셀 중인데, 고래는 조용히 매수 중이다.”
“게시판은 탐욕 상태지만, 퀀트 봇들은 매도를 시작했다.”
게임 루프
가상 시장 시작
AI 에이전트들이 기본 포지션 보유 (의사난수 시드값으로 생성)
이벤트 발생
사용자 생성 또는 자동 생성
에이전트들이 이벤트 해석
매매 발생
게시물 생성
게시물이 다른 에이전트 감정에 영향
가격 변화
사용자가 매매하거나 글로 개입
라운드 종료 후 감정 리포트 제공

핵심 지표
게임 화면에는 이런 지표가 보인다.
시장 공포 지수
시장 탐욕 지수
루머 확산 속도
패닉셀 비율
FOMO 매수 비율
고래 매수/매도 강도
가장 영향력 있는 게시물
가장 많이 흔들린 에이전트
가격 변동 중 감정이 만든 비중

차별점
이 제품은 투자 실력을 키우는 앱이라기보다, 시장을 하나의 감정 생태계로 관찰하는 게임이다.
일반 투자 시뮬레이터가:
가격이 오름 → 매수/매도
라면, 이 제품은:
이벤트 발생 → 감정 전염 → 게시물 확산 → 집단 매매 → 가격 왜곡 → 사용자 개입
을 보여준다.

구현
### 1. 기본 방향

Market Aquarium은 `joonspk-research/generative_agents`의 게임 스택을 활용한다.

기존 프로젝트의 구조는 다음과 같다.

- Django 기반 프론트 서버
- Phaser 3 기반 2D 타일맵 게임 화면
- Python 기반 시뮬레이션 서버
- 에이전트별 상태, 기억, 행동 데이터
- step 단위로 생성되는 movement JSON

Market Aquarium은 이 구조를 그대로 사용하되, 기존의 일상 마을 시뮬레이션을 투자 심리 시장 시뮬레이션으로 재해석한다.

즉, 마을은 하나의 가상 시장이 되고, 에이전트들은 시장 참여자가 된다.

---

### 2. 게임 공간 -> 글로벌이벤트를 동시적으로 알게 할것이다. 반성 x, 감정 전염 x, 분석 x

기존 마을 맵을 새로 만들지 않고, 장소의 의미를 바꿔 사용한다.

| 기존 공간 | Market Aquarium에서의 의미 |
|---|---|
| 각 에이전트의 집 | 개인 판단 공간, 계획 수립 |
| 마을 게시판 / 광장 | 투자 커뮤니티 |
| 거래소 역할 공간 | 매수·매도 실행 장소 |

MVP에서는 맵 자체를 새로 제작하지 않는다.  
대신 에이전트가 어느 장소로 이동하는지에 따라 시장 행동을 표현한다.

예시:

- 집에 있음 → 관망, 계획 수립, 반성
- 게시판으로 이동 → 게시물 작성, 댓글
- 거래소로 이동 → 매수 또는 매도

Market Aquarium은 이 자율 행동 구조를 투자 심리 시뮬레이션에 맞게 변환한다.

5. 자율 행동의 적용 방식
5.1 Daily Planning
장 시작 시 각 에이전트는 자신의 성격에 맞는 투자 계획을 가진다.
예시:
패닉셀 개미:
- 게시판에서 시장 분위기를 확인한다.
- 악재가 나오면 빠르게 거래소로 이동한다.
- 공포 지수가 높아지면 손절한다.

가치투자자:
- 뉴스의 신뢰도를 먼저 확인한다. (뉴스의 신뢰도를 어케 확인하는데?)
- 루머성 악재에는 바로 반응하지 않는다.
- 과매도 상황이면 매수를 고려한다.

음모론 인플루언서:
- 불확실한 이벤트를 찾는다.
- 게시판에 자극적인 해석을 올린다.
- 반응이 크면 추가 게시물을 작성한다.

MVP에서는 LLM으로 매일 계획을 생성하기보다, 에이전트별 템플릿 계획을 사용한다.

5.4 Decide to Talk
에이전트는 마을 게시판에서 글을 쓸지, 댓글을 달지, 반박할지 판단한다. 
예시:
음모론 인플루언서: -> 
"이건 단순 해킹이 아닙니다. 큰손들은 이미 알고 움직였을 가능성이 큽니다."

패닉셀 개미:
"저도 불안해서 일단 팔았습니다..."

퀀트 트레이더:
"아직 근거는 부족하지만, 패닉셀 비율은 비정상적으로 높습니다."

가치투자자:
"신뢰도 낮은 루머에 가격이 과도하게 반응하고 있습니다."

게시물은 다른 에이전트의 감정에 영향을 준다. (llm이 상승정도 판정)
공포성 게시물 → fear 상승
탐욕성 게시물 → greed 상승


5.5 React to Unexpected
돌발 뉴스나 루머가 발생하면 에이전트는 기존 계획을 유지할지 바꿀지 판단한다. (llm이)
예시:
패닉셀 개미:
기존 계획: 게시판을 보며 관망
돌발 이벤트: 거래소 해킹 루머
변경 후 행동: 거래소로 이동해 SELL

매크로 고래:
기존 계획: 관망
돌발 이벤트: 공포 지수 80 돌파
변경 후 행동: 거래소로 이동해 BUY_LARGE

가치투자자:
기존 계획: 신뢰도 확인 후 판단
돌발 이벤트: 신뢰도 낮은 해킹 루머 
변경 후 행동: HOLD
이 기능은 Market Aquarium의 핵심인 “뉴스가 감정을 만들고, 감정이 행동을 바꾸는 과정”을 보여준다.

6. 시뮬레이션 루프
한 라운드는 다음 순서로 진행된다.
1. 장 시작
   → 에이전트별 Daily Plan 설정

2. 이벤트 발생
   → 사용자 입력 또는 시스템 자동 생성

3. 이벤트 중요도 평가
   → 에이전트별 Poignancy Scoring

4. 감정 변화
   → fear / greed 업데이트

5. 게시판 행동
   → 글 작성, 댓글, 반박, 루머 확산

6. 계획 변경
   → React to Unexpected

7. 매매 행동
   → BUY / SELL / HOLD 결정

8. 장소 이동
   → 집 / 게시판 / 거래소로 이동

9. 가격 변화 계산
   → 이벤트 충격 + 감정 전염 + 집단 매매 반영

10. 라운드 리포트
   → 공포/탐욕 지수, 루머 확산, 감정 기여도 표시
   → 게임이 끝나고 overall 보고서를 생성

11. 반성
   → 중요 사건이 누적된 에이전트는 Reflection 생성


7. 가격 변화 모델
실제 오더북을 구현하지 않고, MVP용 단순 가격 엔진을 사용한다.
가격 변화율 =
  이벤트 자체 충격
+ 순매수/순매도 압력(에이전트들의 캐시량에 따라 반영)
+ 감정 과열 압력(에이전트들의 fear&greed)
+ 노이즈 

// 이벤트 - 충격 매핑 되어있어야함

예시:
뉴스 자체 충격: -2.7%
패닉셀 압력: -4.1%
루머 확산 효과: -1.5%
최종 가격 변화: -7.5%

이 모델의 목적은 통제된 환경에서 이슈나 감정의 변화가 가격을 얼마나 왜곡하는지 보여주는 것이다.

9. 화면 구성
게임 화면은 세 영역으로 구성한다.
1. 중앙 2D 맵

2. 좌측 시장 패널 (오버레이 혹은 모달로 구현, 기본적으로는 접혀있게)

   - 공포/탐욕 지수
   - 루머 확산 속도
   - 패닉셀/FOMO 비율
   - 고래 매수/매도 강도
   - 감정 기여도

3. 우측 게시판 피드 (핸드폰 형태)
   - 전체 게시판
   - 종목 별 게시판
   - 댓글
   - 좋아요




예시 화면 상태:
😱 패닉셀 개미 → 거래소 이동, SELL

11. 최종 구현 요약
Market Aquarium은 generative_agents의 자율 행동 구조를 투자 시뮬레이션에 맞게 재해석한다.
기존 NPC의 일일 계획, 중요도 평가, 반성, 대화, 돌발 상황 반응은 각각 다음으로 변환된다.
일일 계획
→ 장 시작 전 투자 계획

중요도 평가
→ 뉴스/루머의 심리적 충격 점수

반성
→ 투자 심리 인사이트 생성

대화
→ 게시판 글, 댓글, 반박

돌발 상황 반응
→ 뉴스에 따른 매매 계획 변경

사용자가 이벤트를 던지면 에이전트들은 각자의 성격과 계획, 기억, 감정 민감도에 따라 다르게 반응한다.
그 반응은 게시판을 통해 전염되고, 집단 매매로 이어지며, 최종적으로 가격을 왜곡한다.
MVP의 목표는 정교한 투자 분석이 아니라 다음 흐름을 플레이 가능한 게임 화면으로 보여주는 것이다.
이벤트 발생
→ 감정 변화
→ 게시판 확산
→ 계획 변경
→ 집단 매매
→ 가격 왜곡
→ 리포트
